# dogs-v-cats-Image-Classification
Deep learning project for dog image classification using Convolutional Neural Networks (CNN) with TensorFlow and Keras.

## Project Overview

This project demonstrates how CNNs can be used for image classification tasks.  
The model is trained on dog image data to learn visual patterns and predict classes with high accuracy.

---

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Google Colab

---

## Features

- Image preprocessing
- CNN model building
- Model training and validation
- Accuracy visualization
- Prediction on test images
- Deep learning-based image classification

---

## Model Architecture

The CNN model includes:

- Convolutional Layers
- MaxPooling Layers
- Flatten Layer
- Dense Layers
- Output Layer

---

## Dataset

The dataset contains dog images used for training and testing the CNN model.

---

## Results

- Achieved high training and validation accuracy
- Successfully classified dog images using CNN

---

## Accuracy Graph

The project includes training and validation accuracy graphs for performance evaluation.
